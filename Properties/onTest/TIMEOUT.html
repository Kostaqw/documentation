TIMEOUT

Количество секунд, разрешенных для выполнения этого теста.

Если это свойство установлено, оно ограничивает выполнение теста, чтобы он не занимал больше указанного количества секунд. Если время выполнения превышает этот лимит, процесс теста будет принудительно завершен, и ctest перейдет к следующему тесту. Этот параметр имеет приоритет над CTEST_TEST_TIMEOUT.

Явное значение 0 означает, что у теста нет тайм-аута, за исключением случаев, когда необходимо учитывать параметр ctest --stop-time.
