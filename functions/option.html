Option

Предоставляет булеву опцию, которую пользователь может выбрать по желанию.

option(<переменная> "<текст_помощи>" [значение])

Если начальное [значение] не предоставлено, по умолчанию используется булево значение OFF. Если <переменная> уже установлена как обычная или кешированная переменная, команда ничего не делает (см. политику CMP0077).

Для опций, которые зависят от значений других опций, см. справку модуля для CMakeDependentOption.

В режиме проекта CMake создается кешированная булева переменная со значением опции. В скриптовом режиме CMake устанавливается булева переменная со значением опции.
