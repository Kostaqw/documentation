set_tests_properties

Устанавливает свойство для тестов.

set_tests_properties(<tests>...
                             [DIRECTORY <dir>]
                             PROPERTIES <prop1> <value1>
                             [<prop2> <value2>]...)

Устанавливает свойство для тестов. Если тест не найден, CMake сообщит об ошибке.

Значения свойств теста могут быть указаны с использованием генераторных выражений для тестов, созданных с помощью подписи add_test(NAME).

Новое в версии 3.28: Видимость может быть установлена в других областях видимости каталогов с использованием следующей опции:

DIRECTORY <dir>

Свойства теста будут установлены в области видимости каталога <dir>. CMake должен уже знать об этом каталоге, либо через вызов add_subdirectory(), либо через указание его в качестве исходного каталога верхнего уровня. Относительные пути рассматриваются как относительные к текущему исходному каталогу. <dir> может ссылаться на бинарный каталог.
