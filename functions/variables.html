CMAKE_CXX_STANDARD

Значение по умолчанию для свойства цели CXX_STANDARD_REQUIRED, если оно установлено при создании цели

CMAKE_CXX_STANDARD_REQUIRED

Значение по умолчанию для свойства цели CXX_STANDARD, если оно установлено при создании цели

CMAKE_CXX_STANDARD_REQUIRED

Значение по умолчанию для свойства цели CXX_STANDARD, если оно установлено при создании цели

CMAKE_CURRENT_SOURCE_DIR

Путь к каталогу исходного кода, который в данный момент обрабатывается. Это полный путь к каталогу исходного кода, который в данный момент обрабатывается CMake. При запуске в режиме сценария cmake -P, CMake устанавливает переменные CMAKE_BINARY_DIR, CMAKE_SOURCE_DIR, CMAKE_CURRENT_BINARY_DIR и CMAKE_CURRENT_SOURCE_DIR на текущий рабочий каталог.

<PROJECT-NAME>_VERSION_MINOR

Вторая компонента номера версии переменной <PROJECT-NAME>_VERSION, установленной командой project()

PROJECT_SOURCE_DIR

Это исходный каталог последнего вызова команды project(), сделанного в текущем каталоге или одном из его родительских каталогов. Обратите внимание, что на него не влияют вызовы project(), сделанные в дочернем каталоге (например, в пределах вызова add_subdirectory() из текущего каталога).
