Ninja

Генерирует файл build.ninja в дерево сборки.

Встроенные цели

all

Зависит от всех целей, необходимых для проекта, за исключением тех, у которых установлен флаг EXCLUDE_FROM_ALL в значение true.

install

Выполняет шаг установки.

install/strip

Новое в версии 3.7: Выполняет установку, за которой следует команда CMAKE_STRIP, если она присутствует.

Переменная CMAKE_STRIP будет содержать утилиту strip платформы, которая удаляет символы информации из сгенерированных бинарных файлов.

install/parallel

Новое в версии 3.30: Создается только если глобальное свойство INSTALL_PARALLEL включено. Выполняет шаг установки для каждого подкаталога независимо и параллельно.

Для каждого подкаталога sub/dir проекта генерируются дополнительные цели:

sub/dir/all

Новое в версии 3.6: Зависит от всех целей, необходимых для подкаталога.

sub/dir/install

Новое в версии 3.7: Выполняет шаг установки в подкаталоге, если он есть.

sub/dir/install/strip

Новое в версии 3.7: Выполняет шаг установки в подкаталоге, за которым следует команда CMAKE_STRIP, если она есть.

sub/dir/test

Новое в версии 3.7: Выполняет шаг тестирования в подкаталоге, если он есть.

sub/dir/package

Новое в версии 3.7: Выполняет шаг упаковки в подкаталоге, если он есть.

Поддержка Fortran

Новое в версии 3.7.

Генератор Ninja условно поддерживает Fortran, если версия инструмента ninja составляет как минимум 1.10 (которая имеет необходимые функции).

Поддержка Swift

Новое в версии 3.15.

Поддержка Swift является экспериментальной, не считается стабильной и может измениться в будущих выпусках CMake.
