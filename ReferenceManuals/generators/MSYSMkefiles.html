MSYS Makefiles

Генерирует makefiles для использования с MSYS (Minimal SYStem) make в оболочке MSYS.

Используйте этот генератор в командной строке оболочки MSYS, используя make как инструмент сборки. Сгенерированные makefiles используют /bin/sh в качестве оболочки для выполнения правил сборки. Они не совместимы с командной строкой Windows.

Для сборки в командной строке Windows используйте генератор MinGW Makefiles.
