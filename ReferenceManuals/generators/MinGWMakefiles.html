MinGW Makefiles

Генерирует makefiles для использования с mingw32-make в командной строке Windows.

Используйте этот генератор в командной строке Windows с MinGW (Minimalist GNU для Windows) в PATH и используя mingw32-make как инструмент сборки. Сгенерированные makefiles используют cmd.exe в качестве оболочки для выполнения правил сборки. Они не совместимы с MSYS или Unix-оболочкой.

Для сборки в оболочке MSYS используйте генератор MSYS Makefiles.
