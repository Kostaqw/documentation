Введение

Генератор CMake отвечает за создание входных файлов для нативной системы сборки. Для сборки проекта необходимо выбрать один из генераторов CMake, чтобы определить, какая нативная система сборки будет использоваться. Дополнительно можно выбрать один из дополнительных генераторов в качестве варианта для некоторых генераторов командной строки, чтобы создать файлы проекта для вспомогательной IDE.

Генераторы CMake зависят от платформы, поэтому каждый может быть доступен только на определённых платформах. Команда cmake(1) с опцией --help выводит список доступных генераторов на текущей платформе. Используйте опцию -G для указания генератора для нового дерева сборки. В cmake-gui(1) можно выбрать генератор интерактивно при создании нового дерева сборки.

Генераторы CMake

Генераторы для командной строки

Эти генераторы поддерживают инструменты сборки для командной строки. Для их использования необходимо запустить CMake из командной строки, окружение которой уже настроено для выбранного компилятора и инструмента сборки.

Makefile Generators

  • Borland Makefiles
  • MSYS Makefiles
  • MinGW Makefiles
  • NMake Makefiles
  • NMake Makefiles JOM
  • Unix Makefiles
  • Watcom WMake

Ninja Generators

  • Ninja
  • Ninja Multi-Config

Генераторы для IDE

Эти генераторы поддерживают файлы проектов для интегрированных сред разработки (IDE). Поскольку IDE сами настраивают своё окружение, CMake можно запускать из любого окружения.

Visual Studio Generators

  • Visual Studio 6
  • Visual Studio 7
  • Visual Studio 7 .NET 2003
  • Visual Studio 8 2005
  • Visual Studio 9 2008
  • Visual Studio 10 2010
  • Visual Studio 11 2012
  • Visual Studio 12 2013
  • Visual Studio 14 2015
  • Visual Studio 15 2017
  • Visual Studio 16 2019
  • Visual Studio 17 2022

Другие Генераторы

  • Xcode

Дополнительные Генераторы

Устарело с версии 3.27: Поддержка "Дополнительных Генераторов" устарела и будет удалена в одной из будущих версий CMake. IDE могут использовать cmake-file-api(7) для просмотра деревьев сборки, созданных CMake.

Некоторые из генераторов CMake, перечисленных в выводе команды cmake(1) --help, могут иметь варианты, которые указывают на дополнительный генератор для вспомогательного инструмента IDE. Имена таких генераторов имеют форму <дополнительный-генератор> - <основной-генератор>. Следующие дополнительные генераторы известны CMake:

  • CodeBlocks
  • CodeLite
  • Eclipse CDT4
  • Kate
  • Sublime Text 2
