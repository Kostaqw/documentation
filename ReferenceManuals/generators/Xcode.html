Xcode

Генерация файлов проекта Xcode.

Изменено в версии 3.15: Этот генератор поддерживает Xcode 5.0 и выше.

Выбор набора инструментов и системы сборки

По умолчанию, Xcode выбирает свою собственную цепочку инструментов. Опция CMAKE_GENERATOR_TOOLSET может быть установлена, возможно, через опцию cmake -T, чтобы указать другой набор инструментов.

Новое в версии 3.19: Этот генератор поддерживает указание набора инструментов с использованием одной из следующих форм:

  • toolset
  • toolset[,key=value]*
  • key=value[,key=value]*

Набор инструментов указывает имя набора инструментов. Выбранное имя набора инструментов передается в переменную CMAKE_XCODE_PLATFORM_TOOLSET.

Пары key=value формируют список опций, разделенных запятыми, для указания специфических для генератора деталей выбора набора инструментов. Поддерживаемые пары:

  • buildsystem=<variant>

Укажите вариант системы сборки, который следует использовать. См. переменную CMAKE_XCODE_BUILD_SYSTEM для допустимых значений.

Например, чтобы выбрать оригинальную систему сборки в Xcode 12, выполните cmake с опцией -T buildsystem=1.

Поддержка Swift

Новое в версии 3.4.

При использовании генератора Xcode с Xcode 6.1 или выше можно включить язык Swift с помощью команды enable_language() или project().

Ограничения

Генератор Xcode не поддерживает источники, зависящие от конфигурации. Такой код приведет к ошибке генерации:

add_executable(MyApp mymain-$<CONFIG>.cpp)
