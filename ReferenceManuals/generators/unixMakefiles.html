Unix Makefiles

Генерирует стандартные UNIX makefiles.

Иерархия UNIX makefiles генерируется в дереве сборки. Используйте любую стандартную программу для сборки в стиле UNIX, чтобы собрать проект с помощью цели all и установить проект с помощью цели install (или install/strip).

Для каждой поддиректории sub/dir проекта будет создан UNIX makefile, содержащий следующие цели:

all

Зависит от всех целей, требуемых для поддиректории.

install

Запускает шаг установки в поддиректории, если он есть.

install/strip

Запускает шаг установки в поддиректории, если он есть, а затем выполняет команду CMAKE_STRIP.

Переменная CMAKE_STRIP будет содержать утилиту strip для данной платформы, которая удаляет информацию о символах из сгенерированных бинарных файлов.

test

Запускает шаг тестирования в поддиректории, если он есть.

package

Запускает шаг упаковки
