CMAKE_MODULE_PATH

Список каталогов, разделенных точкой с запятой, указывающий путь поиска модулей CMake, которые будут загружены командами include() или find_package() перед проверкой модулей по умолчанию, поставляемых с CMake. По умолчанию этот список пуст. Предполагается, что он задается проектом.

Часто проект имеет каталог, содержащий различные *.cmake файлы для помощи в разработке. Добавление такого каталога в CMAKE_MODULE_PATH упрощает их загрузку. Например, файл верхнего уровня проекта CMakeLists.txt может содержать:

list(APPEND CMAKE_MODULE_PATH "${CMAKE_CURRENT_SOURCE_DIR}/cmake")
include(Foo) # Загружает ${CMAKE_CURRENT_SOURCE_DIR}/cmake/Foo.cmake
find_package(Bar) # Загружает ${CMAKE_CURRENT_SOURCE_DIR}/cmake/FindBar.cmake
