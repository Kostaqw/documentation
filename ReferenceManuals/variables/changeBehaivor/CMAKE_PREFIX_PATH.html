CMAKE_PREFIX_PATH

Список каталогов, разделённых точкой с запятой, указывающий установочные префиксы, которые будут искаться командами find_package(), find_program(), find_library(), find_file() и find_path(). Каждая команда добавляет соответствующие подкаталоги (например, bin, lib или include), как указано в её документации.

По умолчанию, этот список пуст. Он предназначен для установки проектом.

Также существует переменная окружения CMAKE_PREFIX_PATH, которая используется как дополнительный список префиксов для поиска.

См. также CMAKE_SYSTEM_PREFIX_PATH, CMAKE_INCLUDE_PATH, CMAKE_LIBRARY_PATH, CMAKE_PROGRAM_PATH и CMAKE_IGNORE_PATH.
