CMAKE_BUILD_TYPE

Определяет тип сборки для генераторов с одной конфигурацией (например, генераторы Makefile или Ninja). Типичные значения включают Debug, Release, RelWithDebInfo и MinSizeRel, но также могут быть определены пользовательские типы сборки.

Эта переменная инициализируется первой командой project() или enable_language(), вызываемой в проекте при первом создании нового дерева сборки. Если задана переменная окружения CMAKE_BUILD_TYPE, используется ее значение. В противном случае при включении языка выбирается значение по умолчанию для конкретной цепочки инструментов. Значение по умолчанию часто представляет собой пустую строку, но обычно это нежелательно, и чаще всего более подходит один из других стандартных типов сборки.

В зависимости от ситуации значение этой переменной может обрабатываться с учетом или без учета регистра. Обсуждение этого и других связанных тем см. в разделе Конфигурации сборки.

О генераторах с несколькими конфигурациями см. CMAKE_CONFIGURATION_TYPES.
