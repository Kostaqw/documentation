Введение

CMake — это инструмент для управления сборкой исходного кода. Изначально CMake был разработан как генератор для различных диалектов Makefile, сегодня CMake генерирует современные системы сборки, такие как Ninja, а также проектные файлы для IDE, таких как Visual Studio и Xcode.

CMake широко используется для языков C и C++, но также может быть использован для сборки исходного кода других языков.

Люди, впервые сталкивающиеся с CMake, могут иметь разные начальные цели. Чтобы узнать, как собирать пакет исходного кода, загруженный из интернета, начните с Руководства по взаимодействию с пользователем. Это руководство детализирует шаги, необходимые для запуска исполняемых файлов cmake(1) или cmake-gui(1), выбора генератора и завершения сборки.

Руководство по зависимостям предназначено для разработчиков, желающих начать использовать стороннюю библиотеку.

Для разработчиков, начинающих проект с использованием CMake, подходящим отправной точкой является Учебное пособие по CMake. Руководство cmake-buildsystem(7) предназначено для разработчиков, расширяющих свои знания по поддержке системы сборки и знакомящихся с целями сборки, которые могут быть представлены в CMake. Руководство cmake-packages(7) объясняет, как создавать пакеты, которые можно легко использовать в сторонних системах сборки на основе CMake.
